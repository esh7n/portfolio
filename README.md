# Weather Website

This is a simple weather application website built with HTML, CSS, and JavaScript. It allows users to search for weather information by city.

## Features

- Current weather display (temperature, humidity, description, emoji)
- Navigation between Home, Service, and About Us pages
- Responsive design (basic)

## Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/<YOUR_USERNAME>/weather-Website.git
    ```
2.  **Navigate to the project directory and open `index.html`:**
    ```bash
    cd weatherapi
    ```
    Then, simply open the `index.html` file in your web browser to view the application.

## Usage

-   On the home page, enter a city name in the input field and press "Get Weather" to see the current weather conditions.
-   Navigate between pages using the header links.
-   Click the "WeatherApp" logo to return to the home page.
